# CareerCompass AI

CareerCompass AI is a beginner-friendly AI-powered career recommendation platform built using Python and Flask.

## Features

- Skill-based job recommendations
- Resume upload
- Resume text extraction
- Skill extraction
- Clean Flask web interface

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JSON

## Future Improvements

- TF-IDF based recommendations
- ATS Resume Score
- Learning Roadmap
- NLP-based skill extraction

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```
